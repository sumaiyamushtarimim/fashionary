# **App Name**: Fashionary

## Core Features:

- Dashboard: Provides an overview of key business metrics including orders, products, revenue, expenses, and profit, all dynamically updated.
- Product Management: Allows adding, editing, and managing products including details like name, description, pricing, and inventory levels.
- Order Management: Facilitates tracking new, processing, and completed orders, managing order details, and updating order status.
- Inventory Management: Manages inventory levels, tracks stock movements with fifo lot sistem, and alerts when stock levels are low.
- Customer Management: Stores and manages customer information, tracks customer interactions, and supports personalized communication.
- User and Staff Management: Controls access levels with staff user roles and permissions management with firebase auth for authentication.
- WooCommerce Integration: Syncs product and order data between the ERP and a WooCommerce store, automating sales data updates.
- Purchase Management: Make Purchase and Payment for All Supplier and Vendors.
- DB and Auth: Must Use Prisma for postgresql DB Connection and Firebase for Auth.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and professionalism.
- Background color: Light blue-gray (#ECEFF1), providing a clean and modern feel.
- Accent color: Bright orange (#FF9800) to highlight important actions and data points.
- Body font: 'PT Sans', a modern, warm sans-serif font.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel; Use Space Grotesk for headlines and 'Inter' for body text.
- Consistent use of line icons across the interface for easy recognition.
- A clean and structured layout with a left sidebar for navigation and a main content area for data display.
- Subtle transitions and animations to enhance user experience, like fading in new data.
